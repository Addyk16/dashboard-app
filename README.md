# dashboard-app
Dashboard Web App made using React JS

## Installation
- Clone the repo in your system.
- Go to dashboard folder
  - `cd ./dashboard`
- Run following Command
  - `npm install`
  - `npm run dev`
- You will get the link for the website :)
